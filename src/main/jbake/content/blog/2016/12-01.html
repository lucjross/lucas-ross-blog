title=IntelliJ IDEA | Replace (migrate) all usages of a class with another date=2016-12-01 type=post tags=blog status=published ~~~~~~

This applies to IntelliJ IDEA 2016.3. For example, to switch from javax.validation.constraints.NotNull to lombok.NonNull:

  1. Go to Refactor > Migrate...
  2. In Package and Class Migration window, click New...
  3. In Edit Migration Map, enter a name, e.g. "javax notnull -> lombok nonnull"
  4. Click Add (+)
  5. Select Class
  6. Enter old name ("javax.validation.constraints.NotNull")
  7. Enter new name ("lombok.NonNull")
  8. OK
  9. Run
  10. In Refactoring Preview, verify the old class was found and click Do Refactor
